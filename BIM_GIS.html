BIM - GIS.

El proyecto BIM - GIS es un proyecto que busca conectar la metodología BIM con la metodología GIS. En esta caso la tecnología usada es solamente Python y es una aplicación de escritorio. El objetivo es el siguiente:
Añadir información BIM a los sistemas GIS
Mediante la aplicación, se vuelca la información de un modelo BIM previamente cargado a una capa GIS. Esto se hace filtrando que objetos del modelo se quieren usar y a que capa quiere el usuario volcar la información del modelo. También permite generar un modelo BIM sencillo a partir de una capa GIS y datos proporcionados por el usuario
Puedes ver el funcionamiento en el siguiente vídeo publicado en mi Linkedin
Crear una capa GIS a partir de un modelo BIM
Un reto para las organizaciones es integrar la metodología BIM en sus sistemas GIS. La información es un denominador común y debe de estar compartida en modelos y sistemas. Este programa permite convertir los objetos elegidos por el usuario de un modelo BIM en nodos de una capa GIS que se puede incluir dentro del sistema GIS. Se vuelca toda la información que queira el usuario
