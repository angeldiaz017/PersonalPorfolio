Comunicación en proyectos.

El proyecto de Comunicación es un proyecto para agilizar la toma de notas y de fotos en las visitas a obra de la empresa. Civile.
  • Backend: Servidor ubuntu con nginx. La respuesta del servidor está programada con el framework Django.
  • Frontend: HTML, CSS y Javascript.
En este caso, los usuarios suben las notas que necesiten, incluyendo fotos. Estas notas pueden tener fotos asociadas (georreferenciadas) y de forma predeterminada son privadas. En caso de que, el usuario quiera, puede compartirlo con los compañeros del proyecto o incluso pueden mencionar a compañeros que están fuera del proyecto.
