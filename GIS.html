Proyectos GIS.

Los proyectos GIS es un proyecto ambicioso que pretende tener en una única consulta web toda la información de los proyectos históricos de la empresa Civile.
  • Backend: Servidor ubuntu con nginx. La respuesta del servidor está programada con el framework Django
  • Frontend: HTML, CSS y Javascript
Toda la información de los proyectos se encuentra alojada en un servidor, en concreto en una base de datos Sqlite. Esta base de datos se alimenta de los KML que los trabajores suben a la plataforma. Toda la información se muestra en una web con un mapa.
Además, el mapa dispone de filtros para poder encontrar los trabajos hechos por nombre o código. Una vez que se seleccione un proyecto, muestra en el mapa los proyectos que se encuentran a 10-25-50 km para poder identificar que proyectos se encuentran cerca del proyecto que se está desarrollando
